## My setup:
- Window Manager: [i3-gaps](https://github.com/Airblader/i3)
- Status bar: [polybar](https://github.com/jaagr/polybar)
- Launcher: [dmenu](https://tools.suckless.org/dmenu/) or/and [Rofi](https://davedavenport.github.io/rofi/)
- Notifications: [dunst](https://github.com/dunst-project/dunst)

## Searching files/text
* Command-line fuzzy finder: [fzf](https://github.com/junegunn/fzf)
* Breadth-first version of find: [bfs](https://github.com/tavianator/bfs)
* Text searching: [ripgrep](https://github.com/BurntSushi/ripgrep)
* Quick access to directories: [fasd](https://github.com/clvv/fasd)

## Media:
* Image viewer: [feh](http://feh.finalrewind.org/)
* PDF viewer: [zathura](https://pwmt.org/projects/zathura/)
* Video player: [mpv](https://mpv.io/)
* Audio player: mpd+mpc+ncmpcpp
* Audio visualizer: [C.A.V.A](https://github.com/karlstav/cava)
* Youtube player/downloader: [mps-youtube](https://github.com/mps-youtube/mps-youtube)

## System stats/monitoring
* Task Manager: [htop](http://hisham.hm/htop/)
* System Monitor: [glances](https://github.com/nicolargo/glances)
* Disk usage analyzer: [ncdu](https://dev.yorhel.nl/ncdu)
* Power management: [tlp](http://linrunner.de/en/tlp/docs/tlp-linux-advanced-power-management.html)
* Monitor/Control CPU temperature: [thermald](https://01.org/linux-thermal-daemon/documentation/introduction-thermal-daemon)

## Version Control tools
* Better git diffs: [diff-so-fancy](https://github.com/so-fancy/diff-so-fancy)
* Text mode interface for git: [tig](https://github.com/jonas/tig)
* Mercurial templates for graph logs: [mercurial-cli-templates](https://bitbucket.org/evagelos/mercurial-cli-templates)

## Chat/Mail
* Chat: [Weechat](https://weechat.org/)
* Email: [mutt](http://www.mutt.org/)

## Generic tools
* File Manager: [ranger](http://ranger.nongnu.org/)
* List files/dirs in a tree-like format: tree
* System lock: xautolock
* Taking screenshots: scrot
* Screen color temperature: [Redshift](https://github.com/jonls/redshift)
* Command line HTTP client: [HTTPie](https://github.com/jkbrzt/httpie)
* Google search: [googler](https://github.com/jarun/googler)
* Simplified man pages: [tldr](https://github.com/lord63/tldr.py)
* Colorized cat: [ccat](https://github.com/jingweno/ccat)

## Terminal
* Emulator: [termite](https://github.com/thestinger/termite)
* Shell: [zsh](http://www.zsh.org/)
	* plugin manager: [zplug](https://github.com/zplug/zplug)
	* highlighting: [Fish shell like syntax highlighting](https://github.com/zsh-users/zsh-syntax-highlighting)
* Multiplexer: [tmux](https://tmux.github.io/)
  * plugin manager: [tmp](https://github.com/tmux-plugins/tpm)
* Theme: [material-shell](https://github.com/carloscuesta/materialshell.git)
* Dircolors: [solarized](https://github.com/seebi/dircolors-solarized) or [nord](https://github.com/arcticicestudio/nord-dircolors)

### Text Editor: [Neovim](https://neovim.io/)
* clipboard handler: [xclip](https://github.com/milki/xclip)
* plugin manager: [vim-plug](https://github.com/junegunn/vim-plug)
